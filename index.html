01 · MCP

MCP

Tools zur Laufzeit

Ein externer Prozess gibt Claude Werkzeuge — eine API anrufen, einen Browser steuern, eine Datenbank lesen.

# Chrome-DevTools als MCP
claude mcp add chrome-devtools \
  -- npx -y chrome-devtools-mcp@latest

02 · Skill

Skill

Anleitung die Claude liest

Eine Markdown-Datei mit Frontmatter. Lädt automatisch, sobald die Beschreibung zur Aufgabe passt.

# Skill-Datei anlegen
~/.claude/skills/my-skill/SKILL.md
 
---
description: Summarize git diff
---
