Claude Code · Erweiterungen

Wenn Claude Code etwas Neues lernen soll, gibt es drei Mechanismen. Sie heißen ähnlich, sie tun ähnliches — und doch sind sie klar verschieden. Diese Seite erklärt in einem Sitz, welcher davon für welche Aufgabe gemacht ist, mit lauffähigen Befehlen und Direktlinks zur offiziellen Doku.

01 · MCP

MCP-Server

Tools zur Laufzeit

Ein externer Prozess gibt Claude Werkzeuge — eine API anrufen, einen Browser steuern, eine Datenbank lesen.

# Chrome-DevTools als MCP
claude mcp add chrome-devtools \
  -- npx -y chrome-devtools-mcp@latest

02 · Skill

Agent Skills

Anleitung die Claude liest

Eine Markdown-Datei mit Frontmatter. Lädt automatisch, sobald die Beschreibung zur Aufgabe passt.

# Skill-Datei anlegen
~/.claude/skills/my-skill/SKILL.md

---
description: Summarize git diff
---
